Description

  • PROV-O grounded ontology for the Global.Church data ecosystem. Models ministry activities, church entities, people groups, unified multi-org assessment (gc:Assessment/gc:AssessmentResult), and data exchange provenance.

  • SKOS concept scheme containing the recognized practice types.

  • Identifies the framework or scale used for an assessment. Each concept represents a distinct assessment methodology owned by a specific organization.

  • SKOS concept scheme for Engagement Strength levels (Dimension 2 of 3D Insight).

  • SKOS concept scheme containing the 12 Engagement Accelerators from the 3D Insight framework.

  • SKOS concept scheme for endorsement status values.

  • SKOS concept scheme for endorsement types.

  • SKOS concept scheme for resource lifecycle stages. Progression: NotStarted → Planned → InProgress → InReview → Available.

  • SKOS concept scheme for resource delivery formats.

  • Content domain classification for ministry resources. Hierarchical — top concepts are broad domains, narrower concepts are specializations.

  • How the resource is used — its pedagogical or ministry function.

Assessment c

IRI https://ontology.global.church/core#Assessment
Description

An evaluation of a people group by an organization using a specific methodology. Assessments answer questions like 'how reached is this group?', 'what resources are available?', or 'what phase of progress has been achieved?' Each assessment is performed at a specific point in time, by a specific organization, using a declared methodology. The methodology identifies what was measured and how. An assessment produces one or more result snapshots containing the actual data points.
